# Edu center

---

## 必要なツール

- node, npm (gitのcommitに必要。ない場合はVSCodeなどのcommitizenプラグインでも代用可能。)
- ruby, jekyll (本サイトを動かすのに必要。)
- git (git pushなどに必要。)

## 起動方法

### パッケージインストール

```
$ bundle install
$ npm install
```

### 開発用サーバーの起動

```
$ npm run dev
```

4000番ポートが開くので、 [http://localhost:4000](http://localhost:4000)にアクセスしてください。
